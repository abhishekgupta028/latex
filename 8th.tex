\begin{definition}[Absolute value function]
The absolute value function can be specified as a two-part definition as follows:
\[
|x| =
\begin{cases}
x & \text{if } x \geq 0 \\
-x & \text{if } x < 0
\end{cases}
\]
\end{definition}
